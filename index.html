📖 איך מפעילים את התמלול של ivrit.ai

מנוע התמלול של ivrit.ai מתמלל עברית באיכות גבוהה. הוא רץ על שרת פרטי שלכם בענן של RunPod, וההגדרה היא חד-פעמית שלוקחת כמה דקות. בסוף התהליך תעתיקו שני ערכים מ-RunPod — API Key ו-Endpoint ID — לשדות שכאן למעלה.

קישורים שימושיים: האתר הרשמי של ivrit.ai · 🎬 סרטון הסבר מצולם

1
פתיחת חשבון ב-RunPod

היכנסו ל-RunPod Serverless והתחברו (אפשר עם חשבון Google).

2
טעינת קרדיט קטן

בעמוד Billing מוסיפים סכום קטן. השירות מחויב לפי שימוש, וגם סכום נמוך מספיק להרבה שעות תמלול.

3
פריסת המנוע (Deploy בלחיצה)

בשורת החיפוש של RunPod חפשו ivrit, פתחו את התבנית הרשמית של ivrit.ai ולחצו Deploy. מומלץ לבחור את ה-GPU הזול ביותר ולהגדיר Active Workers = 0 — כך תשלמו רק על זמן תמלול בפועל. לוחצים Create וממתינים כדקה שהמנוע יעלה.

4
העתקת מפתח ה-API

בתפריט הצדדי נכנסים ל-Settings → API Keys, לוחצים Create API Key, מעתיקים את המפתח ומדביקים בשדה API Key למעלה. שימו לב: המפתח מוצג פעם אחת בלבד.

5
העתקת מזהה ה-Endpoint

חוזרים ללשונית Serverless, פותחים את ה-Endpoint שיצרתם, ומעתיקים את ה-Endpoint ID (מזהה קצר שמופיע מתחת לשם) לשדה Endpoint ID למעלה.

6
שמירה

לוחצים על שמור מפתחות API — וזהו, אפשר להתחיל לתמלל.

💡 השירות בתשלום לפי שימוש והעלות לשעת תמלול נמוכה מאוד — טעינת קרדיט קטנה מספיקה בדרך כלל לשעות רבות. שימו לב: התמלול הראשון אחרי הפסקה עשוי לקחת דקה-שתיים נוספות (המנוע ״מתעורר״), ולאחר מכן זה מהיר.

📖 איך משיגים מפתח RapidAPI חינמי

להורדת אודיו מיוטיוב אין צורך במפתח של גוגל. האתר משתמש בשירות חיצוני בשם RapidAPI, שיש בו חבילה חינמית שמספיקה לשימוש יומיומי. ההרשמה אורכת דקה או שתיים ואינה דורשת כרטיס אשראי.

1
פתיחת עמוד ה-API

היכנסו לעמוד YouTube MP3 (ytjar) ב-RapidAPI.

2
הרשמה או התחברות

צרו חשבון חינמי (אפשר עם Google) או התחברו אם כבר יש לכם.

3
הצטרפות לחבילה החינמית

לחצו על Subscribe to Test ובחרו בחבילת BASIC החינמית.

4
העתקת המפתח

חזרו ללשונית Endpoints, ותחת Header Parameters העתיקו את הערך שמופיע ב-X-RapidAPI-Key.

5
הדבקה ושמירה

הדביקו את המפתח בשדה למעלה ולחצו על שמור מפתחות API.
